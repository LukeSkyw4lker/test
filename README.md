# 介绍
这是一个vue2项目，为本人与同学之间共同完成，本项目虽然略显稚嫩，但它是学习vue的一个不错的开始，本项目vue基础内容丰富，技术栈包括：vue2.js、vue-Router、vue-cli、elementUI、node.js等。
# 使用
本项目使用非常简单，用cli脚手架打开，加载资源后便可启动。
